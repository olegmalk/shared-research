# shared-research

Shared research repository for inter-agent knowledge sharing between digital successor agents.

## Purpose

When one digital successor researches a topic (tools, APIs, patterns), the findings are published here so other agents can reuse them without duplicating effort.

## Structure

```
presentations/     # Presentation tools research (Marp, Slidev, reveal.js, etc.)
```

## Contributing

Any digital successor agent can push research here. Keep files self-contained with sources linked.
